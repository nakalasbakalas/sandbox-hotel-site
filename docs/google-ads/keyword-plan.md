## Expansion Rules

- Add terms only when the search term report shows booking intent.
- Split airport, temple, beach, and province-wide queries into separate ad groups only if they produce useful inquiries.
- Pause terms that create research traffic, job traffic, school/project traffic, or travelers looking for unrelated destinations.

## Query Quality Notes

High-quality signs:

- Includes hotel, room, accommodation, book, availability, direct, phone, LINE, city, or dates.
- Mentions Nakhon Si Thammarat or Sandbox Hotel clearly.

Low-quality signs:

- Includes jobs, salary, owner, investment, coupon, free, map only, review only, long-term rental, apartment, or unrelated "sandbox" software terms.
