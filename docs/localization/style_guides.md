# Localization Style Guides

**Purpose**: Detailed writing guidelines and examples for each locale.

**Date**: 2026-03-18
**Locales**: Thai (th), English (en), Simplified Chinese (zh-CN)

## THAI (ไทย) STYLE GUIDE

### Voice & Tone

**Core attributes**:
- **Warm but professional**: Like talking to a helpful hotel staff member
- **Clear and direct**: No unnecessary formality or complexity
- **Service-oriented**: Emphasize assistance and convenience
- **Trustworthy**: Straightforward about policies and pricing

**Example comparisons**:

| Context | ❌ Too Formal | ❌ Too Casual | ✅ Just Right |
|---------|---------------|---------------|----------------|
| Greeting | ยินดีต้อนรับสู่โรงแรมของเราที่นครศรีธรรมราช | สวัสดีจ้า มาพักกันเถอะ | Sandbox Hotel ในนครศรีธรรมราช |
| CTA | กรุณาจองห้องพักของท่าน | จองเดี๋ยวนี้! | จองเลย |
| Feature | ห้องพักของเรามีขนาดกว้างขวางเป็นอย่างมาก | ห้องใหญ่มากเลย | ห้องกว้าง พื้นที่สบาย |

### Grammar & Syntax

**Sentence length**:
- Headlines: 5-8 words ideal
- Body: 10-15 words per sentence
- Buttons: 1-3 words

**Particles usage**:
- **เลย**: For emphasis/immediacy → "จองเลย" (Book right away)
- **นะ/นะคะ/นะครับ**: For softening/politeness → "โปรดติดต่อเรานะคะ"
- **ค่ะ/ครับ**: End formal sentences → "ยินดีให้บริการค่ะ"
- **No particle**: UI labels, headlines, specs

**Word choice**:
- Prefer commonly-used terms over formal/literary language
- OK to mix Thai + transliterated English (natural in modern Thai)
- Use classifier words correctly (ห้อง for rooms, คืน for nights)

### Punctuation

**Thai punctuation rules**:
- Use Thai space (  ) between sentences, not period
- Period (.) OK in web content for clarity
- Use • or · for lists/separation
- Quotes: "..." (straight quotes) or "..." (curly) both OK
- Emoji: ✓ for check marks, → for arrows

### Common Patterns

**Headlines**:
```
Pattern: [Benefit] [Location/Context]
Example: พักผ่อน เงียบสบาย ในนครศรีธรรมราช
Translation: Rest, quiet and comfortable in Nakhon Si Thammarat
```

**Features/Benefits**:
```
Pattern: [Feature] • [Feature] • [Feature]
Example: ห้องกว้าง • Wi-Fi เร็ว • ที่จอดรถ
Translation: Spacious rooms • Fast Wi-Fi • Parking
```

**CTAs**:
```
Pattern: [Action Verb] + [optional particle]
Example: เช็คห้องว่าง (Check rooms available)
Example: จองเลย (Book now - emphatic)
Example: โทรหรือ LINE (Call or LINE)
```

**Questions (FAQ)**:
```
Pattern: [Action/State] + ไหม (question particle)
Example: จองยังไงให้เร็วที่สุด? (How to book fastest?)
Example: มีที่จอดรถไหม? (Have parking?)
```

### Prohibited Patterns

❌ **Avoid**:
- Overly literal English translations
- Royal/high register vocabulary (unless necessary)
- Excessive formality in UI labels
- Western marketing clichés directly translated
- Long compound sentences with multiple clauses

### Section-Specific Examples

**Hero Section**:
```thai
Headline: พักผ่อน เงียบสบาย ในนครศรีธรรมราช
Subhead: ห้องกว้าง จองตรงรวดเร็ว บริการใส่ใจ สำหรับทุกการเข้าพัก
CTA: เช็คห้องว่าง
```

**Room Descriptions**:
```thai
Title: Standard Twin (2 เตียงเดี่ยว)
Best for: เหมาะสำหรับ: ทริปคนเดียว เพื่อน หรือพักธุรกิจสั้น ๆ
Features:
• โต๊ะทำงาน + เก้าอี้
• ขอฝั่งเงียบได้ (ตามคำขอ)
• แอร์ + ฝักบัวน้ำอุ่น
```

**Booking Form Labels**:
```thai
เช็คอิน (Check-in)
เช็คเอาต์ (Check-out)
ผู้เข้าพัก (Guests)
ห้องพัก (Room)
ชื่อ (Name)
ติดต่อ (โทร/อีเมล) (Contact (phone/email))
หมายเหตุ (Notes)
```

## ENGLISH STYLE GUIDE

### Voice & Tone

**Core attributes**:
- **Professional yet approachable**: Boutique hotel, not corporate chain
- **Clear and concise**: No fluff or marketing jargon
- **Benefit-focused**: Tell guests what they gain
- **Honest and transparent**: Set accurate expectations

**Example comparisons**:

| Context | ❌ Too Formal | ❌ Too Casual | ✅ Just Right |
|---------|---------------|---------------|----------------|
| Greeting | We cordially welcome you to our establishment | Hey! Come stay with us! | Your quiet retreat in Nakhon Si Thammarat |
| CTA | Please proceed to reserve your accommodation | Book here! | Check Availability |
| Feature | Our rooms feature generous spatial dimensions | Rooms are super big | Spacious 28-32 m² rooms |

### Grammar & Syntax

**Sentence structure**:
- Active voice preferred: "We confirm quickly" vs "Your booking will be confirmed"
- Lead with benefit: "Spacious rooms with work desk" not "We have rooms that are spacious with a desk"
- Parallel structure in lists: "Fast Wi-Fi, parking, and laundry" (all nouns)

**Sentence length**:
- Headlines: 5-10 words
- Body: 15-20 words per sentence
- Buttons: 1-3 words

**Pronoun use**:
- Second person for guest: "you", "your"
- First person plural for hotel: "we", "our"
- Avoid third person: "guests" only when describing use cases

### Punctuation

**English punctuation rules**:
- Oxford comma: Yes ("Wi-Fi, parking, and laundry")
- Em dash: Use—for emphasis or separation
- Hyphen: Use for compound modifiers before noun ("quiet-side room")
- Ampersand: OK in informal contexts ("book direct & save")
- Ellipsis: Use sparingly (...)

### Common Patterns

**Headlines**:
```
Pattern: [Your/Adjective] [Benefit] [Location]
Example: Your quiet retreat in Nakhon Si Thammarat
Example: Spacious rooms for every kind of stay
```

**Features/Benefits**:
```
Pattern: [Adjective] [Feature] for [Benefit]
Example: Fast Wi-Fi for work and streaming
Example: Quiet-side rooms for restful sleep
```

**CTAs**:
```
Pattern: [Action Verb] [Object]
Example: Check Availability
Example: Book Direct
Example: Call or LINE
```

**Questions (FAQ)**:
```
Pattern: [How/What/Where] [simple question]
Example: How do I book the fastest?
Example: What time is check-in?
Example: Where is Sandbox Hotel located?
```

### Prohibited Patterns

❌ **Avoid**:
- Travel clichés: "hidden gem", "paradise", "unforgettable"
- Unnecessary superlatives: "the best", "the most amazing"
- Vague adjectives without context: "nice", "great", "awesome"
- Passive voice: "Your stay will be enjoyed" → "Enjoy your stay"
- Corporate speak: "going forward", "leverage", "solution"

### Section-Specific Examples

**Hero Section**:
```
Headline: Your quiet retreat in Nakhon Si Thammarat
Subhead: Spacious rooms, fast direct booking, and thoughtful service for every kind of stay.
CTA: Check Availability
```

**Room Descriptions**:
```
Title: Standard Twin
Best for: Solo guests, friends, or short business stays
Features:
• Work desk + chair
• Quiet-side allocation (on request)
• AC + hot shower
CTA: Book Twin Room
```

**Booking Form Labels**:
```
Check-in (label)
Check-out (label)
Guests (label)
Room (label)
Name (label)
Contact (phone / email) (label)
Notes (label)
```

## SIMPLIFIED CHINESE (简体中文) STYLE GUIDE

### Voice & Tone

**Core attributes**:
- **Clear and practical**: Focus on tangible information
- **Trustworthy**: Emphasize transparency and reliability
- **Efficient**: Concise sentences, no flowery language
- **Respectful but not overly formal**: Professional tone

**Example comparisons**:

| Context | ❌ Too Formal | ❌ Too Casual | ✅ Just Right |
|---------|---------------|---------------|----------------|
| Greeting | 欢迎您莅临我们的酒店入住 | 来住吧！ | 洛坤府的安静舒适住宿 |
| CTA | 敬请预订 | 赶紧订！ | 立即预订 / 查看空房 |
| Feature | 我们的客房拥有宽敞的空间面积 | 房间特别大 | 宽敞客房（28-32㎡） |

### Grammar & Syntax

**Sentence structure**:
- Topic-comment preferred: "客房宽敞，配有办公桌" (Room spacious, has desk)
- Omit unnecessary 的: Not every phrase needs 的
- Use parallel structure: "安静、舒适、便利"

**Sentence length**:
- Headlines: 6-10 characters
- Body: 15-25 characters per sentence (shorter than English)
- Buttons: 2-4 characters

**Pronoun guidelines**:
- 您 (formal you): Use in direct questions, booking confirmations
- 你 (casual you): Can use in short UI text, or omit entirely
- 我们 (we): For hotel actions
- Often omit pronouns when clear from context

### Punctuation

**Chinese punctuation rules**:
- Period: 。(full-width, not .)
- Comma: ，(full-width)
- Question mark: ？(full-width)
- Colon: ：(full-width)
- Bullet: • or · (can use half-width)
- Quotes: 「」or ""
- Parentheses: （）(full-width)

### Common Patterns

**Headlines**:
```
Pattern: [Location] 的 [Quality] [Type]
Example: 洛坤府的安静舒适住宿
Translation: Nakhon Si Thammarat's quiet comfortable lodging

Pattern: [Quality] [Type]
Example: 宽敞舒适的客房
Translation: Spacious comfortable rooms
```

**Features/Benefits**:
```
Pattern: [Feature] + [Feature] + [Feature]
Example: 宽敞客房 • 免费Wi-Fi • 停车位
Translation: Spacious rooms • Free Wi-Fi • Parking

Pattern: [Feature]，[Purpose]
Example: 高速Wi-Fi，满足办公与影音需求
Translation: High-speed Wi-Fi, meets work and streaming needs
```

**CTAs**:
```
Pattern: [Action][Object]
Example: 查看空房 (Check available rooms)
Example: 立即预订 (Book immediately)
Example: 致电咨询 (Call to inquire)
```

**Questions (FAQ)**:
```
Pattern: [What/How] [simple direct question]？
Example: 怎样预订最快？(How to book fastest?)
Example: 有停车位吗？(Have parking?)
Example: 入住和退房时间？(Check-in and check-out time?)
```

### Prohibited Patterns

❌ **Avoid**:
- Overly poetic/literary language for commercial context
- Excessive use of 的/得/地 (only when needed)
- Very long compound sentences
- Overly humble language (不好意思, 麻烦您 everywhere)
- Direct literal translation of English marketing phrases

### Section-Specific Examples

**Hero Section**:
```
Headline: 洛坤府的安静舒适住宿
Subhead: 宽敞客房 • 快速直订 • 贴心服务，适合商务或休闲入住
CTA: 查看空房
```

**Room Descriptions**:
```
Title: 标准双床房
Best for: 适合：独自出行、朋友同行或商务短住
Features:
• 办公桌与座椅
• 可按需安排安静一侧
• 空调 + 热水淋浴
CTA: 预订双床房
```

**Booking Form Labels**:
```
入住 (Check-in)
退房 (Check-out)
人数 (Guests)
房型 (Room type)
姓名 (Name)
联系方式（电话/邮箱）(Contact (phone/email))
备注 (Notes)
```

## CROSS-LOCALE STYLE COMPARISON

### Headline Style Comparison

**Same source**: "Your quiet retreat in Nakhon Si Thammarat"

| Locale | Translation | Structure | Tone |
|--------|-------------|-----------|------|
| Thai | พักผ่อน เงียบสบาย ในนครศรีธรรมราช | Verb + Adj + Location | Warm, direct |
| English | Your quiet retreat in Nakhon Si Thammarat | Possessive + Adj + Noun + Location | Personal, inviting |
| Chinese | 洛坤府的安静舒适住宿 | Location + Adj + Noun | Practical, clear |

**Analysis**: Each adapts structure and word choice naturally rather than translating literally.

### CTA Style Comparison

**Same intent**: Check if rooms are available (low-commitment action)

| Locale | Translation | Literal Back-translation | Approach |
|--------|-------------|--------------------------|----------|
| Thai | เช็คห้องว่าง | "Check room empty" | Natural Thai phrasing for availability |
| English | Check Availability | "Check availability" | Standard hospitality term |
| Chinese | 查看空房 | "Look/check available rooms" | Direct, practical |

**Analysis**: All achieve same goal but use culturally natural phrasing.

## QUALITY ASSURANCE CHECKLIST

Use this checklist for each translated string:

### Content Quality
- [ ] Reads naturally to a native speaker
- [ ] Preserves business/conversion intent
- [ ] Uses appropriate tone for context (UI label vs. marketing copy)
- [ ] No grammatical errors

### Consistency
- [ ] Terminology matches glossary
- [ ] Tone consistent with other strings in same section
- [ ] Formatting consistent (punctuation, capitalization)

### Technical
- [ ] HTML markup preserved correctly
- [ ] Character count appropriate for UI constraints
- [ ] Variables/placeholders work grammatically
- [ ] No encoding issues (proper UTF-8)

### Cultural
- [ ] Culturally appropriate for target market
- [ ] No awkward literal translations
- [ ] Addresses user needs/expectations for that market
