# Archived Duplicate PMS

This directory is a frozen archive of an old duplicate PMS copy that used to live inside `sandbox-hotel-site`.

It is not authoritative, must not be deployed, and must not be used for new PMS or booking-engine work.

Canonical source of truth:
- GitHub repo: `nakalasbakalas/sandbox-pms`
- Local sibling repo: `../sandbox-pms`

If you need to work on reservations, live availability, front desk, housekeeping, cashier, payments, admin, communications, reporting, or security hardening, use `sandbox-pms` instead.

The active `packages/pms` path in this repo is now docs-only on purpose.
